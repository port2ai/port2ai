# port2ai

`port2ai` is a local-first workspace for turning small remote contract work into reliable income.

The project keeps research, offers, client notes, delivery checklists, and reusable templates together so each paid job improves the system for the next one.

## Working Goal

Earn enough from small, bounded remote software and automation contracts to pay for the tools used to find, deliver, and improve that work.

## Starter Offers

- Fix one focused bug in a small web app.
- Clean up, document, and harden an existing script.
- Turn a repeated spreadsheet or manual workflow into a small tool.
- Add tests, CI, or deployment polish to a small repo.
- Improve a simple landing page for speed, clarity, and mobile behavior.

## Repository Map

- `research/`: market notes, platform research, niche ideas, and examples.
- `work/`: active leads, proposals, client notes, delivery checklists, and retrospectives.
- `templates/`: reusable proposal, intake, estimate, and delivery templates.
